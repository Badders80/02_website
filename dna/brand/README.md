# Brand Guidelines

| File | Content |
|------|---------|

## Brand Colors

| Token | Hex | Usage |
|-------|-----|-------|
| `gold-400` | `#d4a964` | Primary accent, CTAs |
| `evolution-black` | `#121212` | Background |
| `evolution-dark` | `#1a1a1a` | Card backgrounds |
| `evolution-gray` | `#2a2a2a` | Borders, dividers |
| `evolution-light` | `#f5f5f5` | Text, foreground |
